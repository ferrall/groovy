Create a new functional component StickingRow that renders a row of sticking cells (one per subdivision).

Props interface (read_first: PlaybackControls.tsx for styling pattern):
- stickingValues: StickingValue[] — Array of current sticking values (from groove.measures[selectedMeasure].sticking)
- onStickingChange: (index: number, value: StickingValue) => void — Callback when a cell value changes
- isActive: boolean — Whether sticking row should be visible/editable
- measureIndex: number — Current measure index (for accessibility labels)

Cell rendering:
- Each cell displays the sticking value (L, R, L/R) or empty string if null
- Cells align exactly with drum grid columns (use same width/padding as grid cells in DrumGridDark)
- Visually distinct from instrument rows (light background, subtle border or separator line)
- Responsive: font size may reduce on small screens (use Tailwind responsive classes)

Interaction — Click-to-cycle (per D-03):
- onClick handler on each cell: cycle to next value in order: Empty → R → L → L/R → Empty
- Immediately calls onStickingChange(index, nextValue)
- No confirmation steps

Interaction — Keyboard shortcuts (per D-04, active when cell focused):
- onKeyDown handler on each cell:
  - 'r' or 'R': set to 'R'
  - 'l' or 'L': set to 'L'
  - '/' or 'b' or 'B': set to 'L/R'
  - 'Backspace' or 'Delete': clear (set to null)
  - 'Space' or 'Enter': cycle to next value
  - All other keys: do nothing (no event.preventDefault)
- All changes apply immediately; no UI flicker

Accessibility (per CLAUDE.md and requirements section 16):
- Each cell has a tabIndex={0} for keyboard navigation
- ARIA label format (read from requirements): "Beat {position} sticking: {value}"
  - Example: "Beat 1 sticking: Right", "Beat 2 and sticking: Left", "Beat 3 sticking: empty"
  - Position calculation: Use beat number + optional "&" for off-beat based on subdivision index
  - Value labels: L="Left", R="Right", L/R="both hands", null="empty"
- ARIA role="group" on the row container
- ARIA label on row: "Sticking setup for measure {measureIndex + 1}"

Styling (per CLAUDE.md — Tailwind + Radix patterns):
- Use Tailwind for layout and spacing
- Cell background: subtle hover effect (slight background color change)
- Cell focus ring: Use Radix UI focus-visible pattern
- Row background: Light gray or subtle color to distinguish from instrument rows
- Padding/margin: Match grid cell heights
- Font: 12px or smaller on mobile (use Tailwind responsive text sizing)

Performance:
- Each cell is a simple button/div (no re-renders on sibling changes due to focused cell)
- No console.log in event handlers (per CLAUDE.md performance constraints)
- useCallback for onStickingChange callbacks to prevent unnecessary re-renders

Return value: The StickingRow component is exported as default. Props type StickingRowProps is also exported.
